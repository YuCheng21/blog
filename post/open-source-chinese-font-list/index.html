名詞解釋

ttc檔 ( TrueType Collection )

字型集合檔案，多個字型 .ttc 組合成一個字體集合文件，單一文件能夠包含多種字體

ttf檔 ( TrueType Font )

單一字型檔，存放一種字型

otf檔 ( OpenType Font )

以 Unicode 為開發基礎，儲存的字型比 TrueType 還要多，螢幕表現也更平滑漂亮，並且可以安裝在 Windows 與 Mac 上
